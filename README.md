# SIVA SILKS — Fashion & Footwear App

## Quick Start

```bash
# 1. Generate platform files (first time only)
flutter create --platforms=android.

# 2. Install packages
flutter pub get

# 3. Run
flutter run -d <device-id>      # Android phone

```
